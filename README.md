A clean, modern, and fully responsive portfolio template built with HTML, CSS, and JavaScript. Perfect for developers, designers, and creatives who want to showcase their work professionally.

## Features
- Dark/Light mode with localStorage
- Smooth scroll animations
- Fully responsive (mobile-first)
- Project filter by category
- Testimonial slider with auto-play
- Contact form with feedback
- Typing effect in hero section
- Back to Top button
- Floating social media icons
- Fast loading (no frameworks)

## Built With
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts (Inter)
